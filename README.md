# Дз 3

- Создать файл `index.html` и подключить JavaScript.
- Объявить переменные с помощью `let` и `const`.
- Вывести значения переменных в консоль.
- Провести эксперименты с изменением `let` и `const`.
- Исправить неправильные названия переменных.
- Создать константы в стиле `SCREAMING_SNAKE_CASE`.
- Собрать строку-визитку с помощью оператора конкатенации `+`.

---
# Что делает программа

1. Подключает файл JavaScript к HTML.
2. Создает переменные с информацией о пользователе:
   - имя;
   - фамилия;
   - возраст;
   - город;
   - хобби;
   - год рождения;
   - любимый цвет.
3. Выводит значения всех переменных в консоль.
4. Демонстрирует разницу между `let` и `const`.
5. Показывает ошибки при изменении `const` и повторном объявлении переменной (в комментариях).
6. Исправляет неправильные названия переменных в соответствии с правилами JavaScript.
7. Создает константы в стиле `SCREAMING_SNAKE_CASE`.
8. Собирает и выводит строку-визитку.
