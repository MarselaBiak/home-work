# Дз 2

- С помощью prompt спросить имя пользователя
- С помощью alert вывести "Hello, John! How are you?" , где John это то, что ввел пользователь

Использовать : prompt, conts, let, use strict

---
# Что делает программа

1. С помощью `prompt` спрашивает имя пользователя.
2. Делает первую букву имени заглавной.
3. С помощью `alert` выводит приветствие, с испольщованием имени пользователя.
4. Выводит приветствие на странице.
